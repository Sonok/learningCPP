## Daily Habit

> Each day: read your target pages -> open the chapter exercise file -> complete the Rapid Fire without hints first -> then attempt at least 1 code-from-scratch problem -> use `<details>` only as a last resort. Track time: reading (~25 min) + exercises (~30 min) = 1 hour/day total.

---

## How to Use Source Citations

Every question in the exercise files includes a citation in the format **(§X.Y, p.ZZ)** — for example, `(§5.2.1, p.53)`.

When you get a question wrong or want deeper understanding:
1. Open the PDF of *A Tour of C++ (2nd Edition)*
2. Jump to the cited page number
3. Re-read the relevant passage
4. Try the question again **before** revealing the `<details>` answer

The citations point to where Stroustrup explains the concept, not where the answer is. This forces active re-reading rather than passive answer-checking.
